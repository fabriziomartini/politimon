# 🏛️ Politimon

**Il simulatore satirico della politica italiana** — gioco browser ispirato a Pokémon Rosso/Blu.

## Come si gioca

1. **Scegli il tuo partito** (FdI, Lega, FI, PD, M5S)
2. **Forma una coalizione** — recluta fino a 3 alleati da altri partiti
3. **Vai alle elezioni** — le tue statistiche + un pizzico di fortuna decidono tutto
4. **Sopravvivi** (o no) alla politica italiana

## Statistiche dei Politici

| Stat | Significato |
|---|---|
| **Carisma** | Capacità di prendere voti |
| **Fedeltà** | Probabilità che non cambino partito |
| **Scandali** | Più alto = più pericoloso per la coalizione |
| **Potere** | Influenza istituzionale |

## Tech Stack

- HTML5 + CSS3 + JavaScript vanilla
- Nessuna dipendenza esterna
- Deploy su GitHub Pages

## Disclaimer

Questo gioco è **pura satira e finzione ludica**. I personaggi, le statistiche e le quote sono inventate a fini comici e non rappresentano posizioni reali delle persone citate.
